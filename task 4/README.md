# Task 4: Optimization Model (Linear Programming)

## 📌 Objective
Solve a business optimization problem using Linear Programming.

---

## ⚙️ Problem
Maximize profit by deciding production quantities of two products.

---

## 🧮 Model

Maximize:
Profit = 40x + 30y

Subject to:
2x + y ≤ 100  
x + y ≤ 80  

---

## 📊 Output

- Optimal values of products
- Maximum profit
- Visualization graph
