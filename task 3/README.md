# 🚀 Titanic Survival Prediction Web App

## 📌 Objective
End-to-end ML project with Flask deployment.

---

## 🔧 Features
- Data preprocessing
- ML model (Logistic Regression)
- Flask web app
- User input form
- Real-time predictions

---

## 🚀 Run Project

### Step 1:
pip install flask pandas scikit-learn joblib

### Step 2:
python train_model.py

### Step 3:
python app.py

---

## 🌐 Open in Browser
http://127.0.0.1:5000/

---

## 🎯 Output
User inputs → Prediction shown on screen

---

## 🛠️ Tech Stack
- Python
- Flask
